响应与测速 ⚡ SmartDNS 核心特色

这是 SmartDNS 区别于传统 DNS 的核心能力：自动测速选出最快的 IP 返回给你，而不是随机或按顺序返回。

推荐 first-ping：ping 最快的 IP 先返回，体验与稳定性兼顾
推荐 ping,tcp:80,tcp:443：ICMP + TCP 综合判断

💡 分流设置

这里对应的是 nameserver 规则，决定"哪个域名走哪个上游组"。匹配顺序从上到下，先匹配先生效。

域名后缀匹配时不需要加点，输入 cn 即生成类似 nameserver /cn/组名 的规则。

新增分流

域名规则 (domain-rules)

对指定域名设置特殊处理：不缓存、禁 IPv6、指定上游组、双栈优选等。

新增域名规则

地址改写与别名 (address / cname)

address：强制域名返回指定 IP，或用 # 屏蔽；cname：域名别名到另一个域名。

IP 规则

对上游返回的 IP 进行过滤：忽略、白名单、黑名单、假冒 NXDOMAIN、IP 别名等。
